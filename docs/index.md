# Welcome to LifeOfPy Documentation

**LifeOfPy** is a modern component registry and developer toolkit for building Python UI applications faster. Similar to `shadcn/ui`, we provide beautifully designed components that you copy and paste directly into your project via our CLI.

## Overview

- **Component Driven**: Stop fighting monolithic UI libraries. Adopt single, focused components into your codebase.
- **Engine First**: The core of LifeOfPy is built on a mathematically robust dependency graph, ensuring conflict-free installations.
- **Agnostic**: We plan to support major Python UI frameworks, starting with CustomTkinter, PyQt, and Textual.

## Quickstart

```bash
# 1. Install the CLI
pip install lifeofpy-cli

# 2. Initialize in your project
lifeofpy init

# 3. Add a component
lifeofpy add button-modern
```
